## 🎯 Desafio da Aula
**Enriqueça o seu perfil:**
1. Adicione uma foto sua (ou um avatar) abaixo do título principal.
2. No seu parágrafo de apresentação, deixe sua profissão/curso em negrito e uma característica sua em itálico.
3. Adicione uma linha divisória para separar o cabeçalho do resto do conteúdo.
4. Crie uma nova lista, mas desta vez numerada, com seus 3 principais objetivos profissionais.
5. Altere o link do seu GitHub para que ele abra em uma nova aba.

## 🏷️ Dicionário de Tags e Atributos

* `<img>`: Insere uma imagem na página. É uma tag que não possui fechamento (self-closing).
    * `src="..."`: (Source) O caminho ou URL de onde a imagem será carregada.
    * `alt="..."`: (Alternative Text) Texto lido por leitores de tela para deficientes visuais e exibido caso a imagem quebre. Essencial para acessibilidade.
* `<strong>`: Deixa o texto em negrito, indicando forte importância semântica.
* `<em>`: Deixa o texto em itálico (Emphasis), indicando ênfase na pronúncia/leitura.
* `<h2>`: Subtítulo de segundo nível. Usado para dividir seções da página.
* `<hr>`: Linha horizontal (Horizontal Rule). Cria uma quebra temática visível.
* `<ol>`: Lista ordenada (Ordered List). Cria listas numeradas (1, 2, 3...).
* `target="_blank"`: Atributo usado na tag `<a>` para forçar o navegador a abrir o link em uma nova aba/janela, retendo o usuário no seu site.
