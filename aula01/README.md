# Aula 01 - Estrutura Básica e Primeiras Tags

Nesta aula, criamos o esqueleto de uma página HTML e aprendemos as tags essenciais para renderização de texto e links.

## 🎯 Desafio da Aula
**Crie o seu perfil base:**
1. Monte a estrutura básica de um documento HTML5.
2. Coloque seu nome como título principal da página.
3. Escreva um parágrafo curto sobre quem é você.
4. Crie uma lista (com bolinhas) de 3 tecnologias que você quer aprender.
5. Adicione um link que leve para o seu GitHub ou LinkedIn.

---

## 🏷️ Dicionário de Tags Utilizadas

* `<html>`: A raiz de todo o documento. Tudo fica dentro dela.
* `<head>`: Cabeçalho do documento (invisível para o usuário). Guarda metadados, links de estilo e o título da aba.
* `<meta charset="UTF-8">`: Define a codificação de caracteres, permitindo o uso de acentos (ç, ã, õ) sem quebrar o texto.
* `<title>`: O nome que aparece na aba do navegador.
* `<body>`: O corpo do documento. Tudo o que é visualmente renderizado na tela fica aqui.
* `<h1>`: Título principal (Heading 1). Vai do `<h1>` ao `<h6>`. O `h1` deve ser usado apenas uma vez por página por questões de Search Engine Optimization (SEO) Otimização para Mecanismos de Busca.
* `<p>`: Parágrafo. Usado para blocos de texto contínuo.
* `<ul>`: Lista não ordenada (Unordered List). Cria listas com marcadores (bolinhas).
* `<li>`: Item da lista (List Item). Vai sempre dentro de uma `<ul>` ou `<ol>`.
* `<a>`: Âncora (Anchor). Cria um link. Exige o atributo `href="URL"` para saber para onde direcionar o clique.
